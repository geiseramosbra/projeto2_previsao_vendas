# \# Projeto de Previsão de Vendas

# 

# Este projeto usa dados de vendas de casas para criar um modelo de previsão de preço. Utilizamos Python, pandas, sklearn, e outras ferramentas para análise, limpeza de dados, modelagem e avaliação.

# 

# \## Como usar

# 

# 1\. Abra o arquivo `house-prices-advanced-regression-techniques.ipynb` no Jupyter Notebook.

# 2\. Execute as células passo a passo para entender o processo.

# 3\. O modelo usa regressão linear para prever os preços das casas.
